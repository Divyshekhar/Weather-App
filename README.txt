1) You need an active internet connection to get the desired location's weather.

2) You also need to download simple.json.1.1.1.jar file from https://storage.googleapis.com/google-code-archive-downloads/v2/code.google.com/json-simple/json-simple-1.1.1.jar

3) You need to add dependencies and path to the your code editor.

4) Then adjust the assets address to the code.(else the images won't show up)

5) Run the AppLauncher to get see the app.

6) Type any location's name and click on the search button to see the result.
